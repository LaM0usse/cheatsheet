Énumération

Nmap

▼
nmap -sV -Pn TARGET

Scan rapide des ports

nmap -p- --min-rate 5000 -Pn -oN TARGET

Scan rapide des TOUS les ports

nmap -A -sC -T4 -n -Pn -p 22,80,8443 TARGET

Scan détaillé sur des ports spécifiques

WhatWeb

▼
whatweb URL

Identification des technologies web (CMS, frameworks, serveurs)

Curl

▼
curl -I URL

Récupérer les en-têtes HTTP pour identifier les techno (Apache, Nginx)

curl -X OPTIONS -v URL/uploads/

Chercher les méthodes HTTP autorisées (si PUT -> essayer d'upload un shell)

curl -T shell.php URL/uploads/

Upload d'un shell sur le serveur

Dirb

▼
dirb URL DIRLIST

Scan de répertoires classique

dirb URL -X .php,.txt,.html

Recherche avec extensions spécifiques

Ffuf

▼
ffuf -u URL/FUZZ -w DIRLIST

Fuzzing rapide de répertoires

ffuf -u URL/FUZZ -w DIRLIST -fc 404,403

Filtrer les codes de réponse 404 et 403

ffuf -u URL/?FUZZ=test -w WORDLIST -fs 1234

Fuzzing de paramètres GET en filtrant par taille de réponse

ffuf -u URL/FUZZ -e .zip,.bak,.php,.txt,.js,.php.bak,.php.swp,.sql,.env,.config,.old -w WORDLIST -fc 404,403 -ic

Fuzzing d'extensions de fichiers

Hakrawler

▼
echo URL | hakrawler

Crawler simple pour découvrir les liens et endpoints (-u pour unique à la fin)

echo URL | hakrawler -js -subs

Extraction des fichiers JS et sous-domaines

Gobuster

▼
gobuster dir -u URL -w DIRLIST

Bruteforce de répertoires

gobuster dir -u URL -w DIRLIST -x php,html,txt,js -t 50 -b 404,403

Recherche avec extensions de fichiers spécifiques et 50 threads en excluant les codes 404 et 403

Wfuzz

▼
wfuzz -c -w DIRLIST URL/FUZZ

Fuzzing de répertoires avec couleurs

wfuzz -c -w WORDLIST --hc 404 URL?param=FUZZ

Fuzzing de paramètres GET en cachant les 404

wfuzz -c -w WORDLIST -d "username=admin&password=FUZZ" URL/login

Bruteforce de mot de passe en POST
