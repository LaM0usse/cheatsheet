Énumération

Nmap

▼
nmap -p- --min-rate 10000 TARGET

Scan rapide de tous les ports (65535)

nmap -sC -sV -O -p22,80,443 TARGET

Scan détaillé sur ports spécifiques
-sC : Scripts par défaut (détecte les accès anonymes FTP, partages SMB...)
-sV : Détection des versions exactes (crucial pour chercher des exploits sur SearchSploit)
-O : Détection de l'OS

nmap -sU -sV --top-ports 20 TARGET

Scan UDP des 20 ports les plus courants

nmap -sC -sV -O -p- --min-rate 5000 -oN nmap.txt TARGET

Scan complet avec tous les ports, scripts, versions et OS + sauvegarde

WhatWeb

▼
whatweb URL

Identification des technologies web (CMS, frameworks, serveurs)

whatweb -v URL

Mode verbose avec plus de détails

whatweb -a 3 URL

Agressivité niveau 3 (analyse approfondie)

WhoIs

▼
whois TARGET

Informations sur le propriétaire du domaine/IP

whois URL | grep -i "name\|email\|phone"

Extraire les contacts du domaine

Gobuster

▼
gobuster dir -u URL -w DIRLIST

Bruteforce de répertoires

gobuster dir -u URL -w DIRLIST -x php,html,txt,js

Recherche avec extensions de fichiers spécifiques

gobuster dir -u URL -w DIRLIST -t 50 -b 404,403

50 threads en excluant les codes 404 et 403

Feroxbuster

▼
feroxbuster -u URL -w DIRLIST

Bruteforce récursif automatique

feroxbuster -u URL -w DIRLIST -x php,txt,html -t 100

Recherche rapide avec extensions et 100 threads

feroxbuster -u URL -w DIRLIST --depth 3 --filter-status 404

Récursion niveau 3 en filtrant les 404

Ffuf

▼
ffuf -u URL/FUZZ -w DIRLIST

Fuzzing de répertoires (très rapide)

ffuf -u URL/FUZZ -w DIRLIST -fc 404,403

Filtrer les codes de réponse 404 et 403

ffuf -u URL/?FUZZ=test -w WORDLIST -fs 1234

Fuzzing de paramètres GET en filtrant par taille de réponse

ffuf -u URL -H "Host: FUZZ.domain.com" -w WORDLIST

Fuzzing de sous-domaines (VHOST)

Dirb

▼
dirb URL DIRLIST

Scan de répertoires classique

dirb URL -X .php,.txt,.html

Recherche avec extensions spécifiques

Wfuzz

▼
wfuzz -c -w DIRLIST URL/FUZZ

Fuzzing de répertoires avec couleurs

wfuzz -c -w WORDLIST --hc 404 URL?param=FUZZ

Fuzzing de paramètres GET en cachant les 404

wfuzz -c -w WORDLIST -d "username=admin&password=FUZZ" URL/login

Bruteforce de mot de passe en POST

Hakrawler

▼
echo URL | hakrawler

Crawler simple pour découvrir les liens et endpoints

echo URL | hakrawler -d 3 -u

Profondeur 3 avec URLs uniques uniquement

echo URL | hakrawler -js -subs

Extraction des fichiers JS et sous-domaines

SMB Enumeration

▼
smbclient -L //TARGET -N

Liste les partages SMB sans authentification

enum4linux -a TARGET

Énumération complète SMB (utilisateurs, partages, groupes)

Énumération DNS

▼
dig axfr @TARGET domain.com

Tentative de transfert de zone DNS

dnsenum --dnsserver TARGET domain.com

Énumération DNS complète

Énumération LDAP

▼
ldapsearch -x -H ldap://TARGET -b "DC=domain,DC=com"

Requête LDAP basique
