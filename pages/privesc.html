Élévation de privilèges

Linux - SUID/SGID

▼
find / -perm -4000 -type f 2>/dev/null

Recherche de binaires SUID

find / -perm -2000 -type f 2>/dev/null

Recherche de binaires SGID

Linux - Capabilities

▼
getcap -r / 2>/dev/null

Liste les capabilities des binaires

Linux - Sudo

▼
sudo -l

Liste les commandes sudo autorisées

sudo -u USER /bin/bash

Exécuter bash en tant qu'un autre utilisateur

Linux - Cron Jobs

▼
cat /etc/crontab

Afficher les tâches cron système

ls -la /etc/cron*

Lister les répertoires cron

Windows - Tokens

▼
whoami /priv

Afficher les privilèges du token

Windows - Services

▼
sc query

Lister les services Windows

sc qc "ServiceName"

Configuration d'un service spécifique

./linpeas.sh
